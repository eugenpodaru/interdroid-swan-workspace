** ContextDroid **
------------------

ContextDroid is a framework for collection and querying sensor data
for Android powered smartphones.


What Is ContextDroid?
---------------------

This project aims to make it easy to build context aware applications
for the Android mobile operating system. The goal is to make it easy
for application developers to write context aware applications while
minimizing the energy usage of monitoring context as much as possible.
